Часто задаваемые
вопросы

Нужно ли платить за AI? ▶
Нет. Gemini API от Google предоставляет бесплатный ключ с лимитом 1500 запросов в день. Этого хватает для активной разработки. Ключ получается на aistudio.google.com за 1 минуту.
Где хранятся мои проекты? ▶
Проекты сохраняются в localStorage браузера — то есть только на твоём устройстве. Никаких серверов, никаких утечек. В Pro плане появится облачное хранилище.
Какие языки поддерживает редактор? ▶
Подсветка синтаксиса для HTML, CSS, JavaScript, JSON, Markdown. AI может генерировать код на любом языке, но Preview показывает только web-проекты.
Можно ли экспортировать и задеплоить сайт? ▶
Да! Кнопка Export скачивает ZIP-архив со всеми файлами проекта. Распакуй и загрузи на любой хостинг: Netlify, Vercel, GitHub Pages, и т.д.
Работает ли на мобильных устройствах? ▶
Редактор оптимизирован для десктопа. На мобильных устройствах интерфейс адаптируется, но для комфортной разработки рекомендуется использовать компьютер.
